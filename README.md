# 您的终端助手——Sebastian
- 基于OpenAI Agents SDK + Typer框架搭建的轻量化CLI助手，通过接入您的API_KEY，它可以成为住在终端里的得力助手，项目完全开源，Star是我最大的动力
- 具体功能如下（部分功能正在开发中）：
  - 文件管理：对文件系统对象的查看/创建/删除/移动/重命名/复制/查找/压缩/解压，以及对文件内容的读取/修改
  - 网络功能：公网实时信息搜索/网页内容抓取/网络资源下载/网络流量分析
  - 代码执行/数学计算：提供沙箱环境执行Shell命令，安全运行未知代码脚本
  - Git管理：通过内置的MCP服务轻松帮你管理Git仓库，支持常用的Git操作
  - 邮件管理：通过IMAP/SMTP协议收发邮件，支持邮件的发送，分类，搜索和自动回复
  - 日历和提醒：管理日程安排，设置提醒事项
  - RAG系统：搭建本地知识库，管理您的个人文档
